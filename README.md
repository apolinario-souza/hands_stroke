# Manual Asymmetry and Motor Control After Stroke

## Overview

This study investigated manual asymmetries and motor control strategies in individuals with chronic unilateral stroke. The objective will to determine whether the side of the cerebral lesion (left or right hemisphere) influences motor performance and control mechanisms during a goal-directed aiming task.

## Participants

* No stroke (NS): in progress
* Left Hemisphere Damage (LS): in progress 
* Right Hemisphere Damage (RS): in progress

## Experimental Task

Participants performed a computerized aiming task using a digitizing tablet. Movements were executed with both hands toward a target positioned 21 cm away at 45°.

## Main Variables

### Motor Performance

* Reaction Time (RT)
* Movement Time (MT)
* Response Time (RespT)
* Radial Error (RE)

### Motor Control

* Peak Velocity (PV)
* Relative Time to Peak Velocity (RTPV)
* Number of Corrections (NC)
* First Submovement Error (ER1Sub)
* Second Submovement Acceleration Area (AreaAccel2Sub)

## Keywords

Stroke; Manual Asymmetry; Motor Control; Aiming Task; Hemispheric Specialization; Motor Performance.
